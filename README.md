## VSCode Copilot Network Error Troubleshooting

### Error: ERR_CONNECTION_CLOSED

If you encounter the error message:
```
Sorry, a network error occurred. Please try again later.
Reason: Please check your firewall rules and network connection then try again. 
Error Code: net::ERR_CONNECTION_CLOSED
```
*Note: The actual error message may appear in your VSCode locale language (e.g., Chinese: "抱歉，出现网络错误。请稍后重试。")*

This is a network connectivity issue between VSCode and the GitHub Copilot service. Here are the solutions:

#### Common Solutions

1. **Check Network Connection**
   - Ensure you have a stable internet connection
   - Try disconnecting and reconnecting to your network
   - If using VPN, try disconnecting it temporarily

2. **Firewall Configuration**
   - Add exception for VSCode and GitHub Copilot domains:
     - `*.github.com`
     - `*.githubusercontent.com`
     - `copilot-proxy.githubusercontent.com`
     - `api.github.com`
   - Allow outbound HTTPS traffic (port 443)

3. **Proxy Settings**
   - If behind a corporate proxy, configure VSCode proxy settings:
     ```json
     {
       "http.proxy": "http://proxy.company.com:port",
       "http.proxyStrictSSL": false
     }
     ```
   - Add to VSCode settings.json (File > Preferences > Settings > search "proxy")

4. **Certificate Issues**
   - Some corporate networks use SSL inspection
   - Try setting: `"http.proxyStrictSSL": false` in VSCode settings
   - Or import your company's SSL certificate

5. **VSCode Extension Reset**
   - Disable GitHub Copilot extension
   - Reload VSCode
   - Re-enable the extension
   - Sign in again

6. **Clear VSCode Cache**
   ```bash
   # Windows
   %APPDATA%\Code\User\globalStorage\github.copilot
   
   # macOS
   ~/Library/Application Support/Code/User/globalStorage/github.copilot
   
   # Linux
   ~/.config/Code/User/globalStorage/github.copilot
   ```
   - Delete the copilot folder
   - Restart VSCode

7. **Check GitHub Copilot Status**
   - Visit [GitHub Status](https://www.githubstatus.com/)
   - Check if Copilot service is experiencing issues

8. **Update Software**
   - Update VSCode to the latest version
   - Update GitHub Copilot extension
   - Restart your computer

#### Network Requirements

GitHub Copilot requires access to:
- `https://api.github.com` (port 443)
- `https://copilot-proxy.githubusercontent.com` (port 443)
- `https://*.githubusercontent.com` (port 443)

#### Still Having Issues?

1. Check VSCode Output panel: View > Output > GitHub Copilot
2. Enable logging in settings:
   ```json
   {
     "github.copilot.advanced": {
       "debug.overrideEngine": "gpt-4",
       "debug.showScores": true
     }
   }
   ```
3. Report issue to [GitHub Copilot Support](https://support.github.com/)
